## Local Setup using npm or Docker
<details>
<summary>‚öõÔ∏è npm Setup</summary>
If you prefer to use npm, follow these steps:

1. Make sure you have Node.js and npm installed on your machine.
2. Navigate to the root of tensormap-client where **package.json** is present
3. Install the required dependencies by running: `npm install`
4. Start the development server with:` npm start`
5. Open a web browser and go to http://localhost:3000 to access the application.
</details>

<details>
<summary> üê≥ Docker Setup </summary>
Make sure you have the latest version of Docker installed then follow these steps:

1. Navigate to where the Dockerfile is located
2. Build the Docker image with: `docker build -t tensormap-client .`
3. Run the Docker container:    `docker run -p 3000:3000 tensormap-client`
4. Open web browser and go to http://localhost:3000 to access the application.
</details>
